# Publication Repository

Web application generated by [BESSER](https://github.com/BESSER-PEARL/BESSER).

## About

This application provides a simple and automated repository for managing research publications. It enables research groups to organize and track their publications while offering useful metrics and visual insights. The platform aggregates key indicators such as the total number of publications, total citations, number of authors, and number of institutions involved. It also provides visualizations that help users explore the distribution and trends of these metrics.

This application was created using the BESSER Web Modeling Editor, a low-code open-source platform for building software.

## How to use

Use the left menu to navigate through the available features:

- **Home**:
    - View global metrics and visualizations across all publications.

- **Publications**:  
    - Manually manage publications (conference papers, journal articles, theses, proceedings, books, and others).  
    - Automatically import publications from BibTeX.  
    - Retrieve citation counts automatically using OpenAlex.  
    - Visualize the most cited publications.  
    - Explore the yearly growth of publications.

- **Institutions**:  
    - Manage institutions.  
    - Visualize the institutions with the highest number of affiliated authors in the publications.  
    - Visualize the institutions with the highest number of publications.

- **Authors**:  
    - Manage authors.  
    - Visualize authors with the most publications.  
    - Visualize authors with the highest number of citations.

## Local Development

### Docker (Full Stack)
```bash
docker-compose up
# Frontend: http://localhost:3000
# Backend: http://localhost:8000
```

### Frontend
```bash
cd frontend
npm install
npm start
# Opens at http://localhost:3000
```

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main_api:app --reload
# API at http://localhost:8000
```
